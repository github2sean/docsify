# Welcome to SeanDocs




## 使用教程

- npm i docsify-cli -g   快速安装  
- docsify init ./docs 初始化目录

```
  index.html 入口文件
  README.md 会做为主页内容渲染
  .nojekyll 用于阻止 GitHub Pages 会忽略掉下划线开头的文件
```
- docsify serve 运行命令

- https://docsify.js.org/ 官网
